## Технические требования

### Code Quality
- ESLint с @vue/eslint-config-typescript
- Prettier для форматирования
- TypeScript strict mode включен
- Unit тесты для composables и stores

### Performance Tips
- Использовать v-memo для дорогих списков
- Lazy load маршрутов и тяжелых компонентов
- Использовать shallowRef/shallowReactive для больших структур данных
- Избегать ненужных watchers
- Использовать computed свойства вместо методов в шаблонах
